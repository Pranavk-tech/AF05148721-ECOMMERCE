# AF05148721-ECOMMERCE
Ecommerce is a full-featured web application that simulates an online shopping platform. It allows users to browse products, add items to a cart, make purchases, and track orders. The system also provides an admin interface to manage products, categories, and orders, making it a complete solution for online retail.

Features
User Authentication: Secure registration and login for customers and admins.
Product Management: Add, edit, or remove products with categories and pricing.
Shopping Cart: Add, remove, and update items in the cart before checkout.
Order Management: Place orders, track status, and view order history.
Admin Dashboard: Manage products, categories, and monitor all transactions.

Technologies Used
Frontend: HTML, CSS, JavaScript (or React/Angular if used)
Backend: Node.js / Python / Java / PHP
Database: MySQL / PostgreSQL / MongoDB
